# Portfolio — dhanzz

Static portfolio siap deploy (GitHub Pages / Vercel). Tanpa build step — cukup upload.

## Struktur
- `index.html` — halaman utama
- `styles.css` — styling modern (dark/light)
- `script.js` — AOS + toggle tema
- `assets/svg/logo-placeholder.svg` — ganti dengan SVG logo kamu
- `assets/img/*` — gambar proyek (placeholder)

## Cara Pakai
1. Ganti teks profil & link sosial di `index.html`.
2. Taruh logo SVG kamu di `assets/svg/` lalu ubah `src` logo.
3. Update gambar project (format `.webp` direkomendasikan).
4. (Opsional) Ubah action Formspree di form kontak.
5. Deploy:
   - **GitHub Pages**: push repositori → aktifkan Pages.
   - **Vercel**: import dari GitHub → deploy.
